# Python Weather CLI

A simple command-line weather application built with Python and the OpenWeather API.

## Features
- Fetch current weather by city
- CLI-based (no UI)
- Uses OpenWeather API
- Console logging

## Requirements
- Python 3.8+
- `requests` library
- OpenWeather API key
